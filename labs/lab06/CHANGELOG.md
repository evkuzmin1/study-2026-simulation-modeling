# Changelog

Все значимые изменения проекта документируются в этом файле.

Формат основан на принципах Keep a Changelog.

### Added

- Добавлена лабораторная работа №6 по теме «Реализация основных моделей в подходе сетей Петри».
- Реализована модель SIR в подходе сетей Петри:
  - состояния `S`, `I`, `R`;
  - переход заражения `infection: S + I → I + I`;
  - переход выздоровления `recovery: I → R`;
  - начальная маркировка `[990.0, 10.0, 0.0]`.

- Добавлен исходный модуль модели:

  `labs/lab06/scripts/src/SIRPetri.jl`

- В модуле реализованы функции:
  - построения SIR-сети Петри;
  - детерминированной симуляции;
  - стохастической симуляции по алгоритму Гиллеспи;
  - построения графиков динамики `S`, `I`, `R`;
  - подготовки визуального представления сети.

- Добавлен базовый скрипт:

  `labs/lab06/scripts/sirpetri_run.jl`

  Скрипт выполняет детерминированную и стохастическую симуляции SIR-модели, сохраняет CSV-файлы и строит графики динамики.

- Добавлен скрипт сканирования параметра заражения:

  `labs/lab06/scripts/sirpetri_scan_parameters.jl`

  Скрипт перебирает значения `β`, рассчитывает максимальное число инфицированных `peak_I` и конечное число выздоровевших `final_R`.

- Добавлен скрипт построения анимации:

  `labs/lab06/scripts/sirpetri_animate.jl`

  Скрипт формирует GIF-анимацию изменения численности групп `S`, `I`, `R` во времени.

- Добавлен итоговый отчётный скрипт:

  `labs/lab06/scripts/sirpetri_report.jl`

  Скрипт строит графики сравнения детерминированной и стохастической динамики, а также график чувствительности пика инфицированных к параметру `β`.

- Для всех основных скриптов добавлены literate-версии:
  - `sirpetri_run_literate.jl`;
  - `sirpetri_scan_parameters_literate.jl`;
  - `sirpetri_animate_literate.jl`;
  - `sirpetri_report_literate.jl`.

- Для всех основных скриптов добавлены параметризованные версии:
  - `sirpetri_run_params.jl`;
  - `sirpetri_scan_parameters_params.jl`;
  - `sirpetri_animate_params.jl`;
  - `sirpetri_report_params.jl`.

- Добавлены результаты моделирования:
  - CSV-файлы с результатами детерминированных и стохастических запусков;
  - графики динамики SIR-модели;
  - графики параметрических экспериментов;
  - GIF-анимации;
  - итоговые отчётные графики.

- Добавлен итоговый Quarto-отчёт:

  `labs/lab06/report/simulation-modeling-lab06-report.qmd`

- Добавлена Beamer-презентация:

  `labs/lab06/report/simulation-modeling-lab06-presentation.qmd`

- Добавлен BibTeX-файл со списком литературы:

  `labs/lab06/report/bib/cite.bib`

### Changed

- Расширена структура проекта для материалов лабораторной работы №6.
- Сохранён общий workflow предыдущей лабораторной работы:
  - обычный скрипт;
  - literate-версия;
  - генерация notebook;
  - параметризованная версия;
  - итоговый отчёт;
  - презентация.

- Отчёт оформлен по аналогии с ранее засчитанной лабораторной работой:
  - рисунки вставлены по ходу выполнения;
  - скриншоты сопровождаются пояснениями;
  - листинги указаны рядом с соответствующими этапами;
  - графики сопровождаются интерпретацией.

- Параметризованные версии скриптов сделаны компактными:
  - выбраны только содержательные сценарии;
  - количество графиков ограничено;
  - результаты не перегружают отчёт.

- Для презентации настроена сборка в формате Beamer.
- Для отчёта и презентации добавлены настройки, отключающие автоматическое обновление TeX Live/TinyTeX при генерации PDF.
